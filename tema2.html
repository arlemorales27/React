Guía práctica para construir aplicaciones web con Styled Components

Duración: 18 secciones

Construcción de una Aplicación de Tareas con React y Emotion

1. Instalar emotion/react y emotion/styled

¿Qué es Styled Components? Styled Components es una librería que permite escribir estilos CSS directamente en componentes React usando JavaScript. Usaremos @emotion/styled, una alternativa popular a la librería styled-components original, y @emotion/react para soporte de temas y otras características avanzadas.

Paso 1: Instalar las dependencias

En el directorio del proyecto (my-react-app), ejecuta:

# Instalar dependencias
npm install @emotion/react @emotion/styled

Esto instala:

  • @emotion/react: Proporciona utilidades como soporte para temas y el componente css.
  • @emotion/styled: Permite crear componentes estilizados con sintaxis similar a CSS.

Paso 2: Verificar la instalación

Asegúrate de que las dependencias se agregaron a package.json:

{
  "dependencies": {
    "@emotion/react": "^11.11.1",
    "@emotion/styled": "^11.11.0",
    // otras dependencias
  }
}

2. Creación Styled Título

¿Qué es un Styled Component? Un styled component es un componente React con estilos asociados directamente, definidos junto al componente en lugar de usar archivos CSS separados.

Ejemplo Práctico

Estilizaremos el título del componente Header usando @emotion/styled. Modifica src/components/Header.jsx:

// src/components/Header.jsx
import React from 'react';
import styled from '@emotion/styled';

const StyledTitle = styled.h1`
  font-size: 2.5rem;
  color: #2c3e50;
  text-align: center;
  margin-bottom: 0.5rem;
  text-transform: uppercase;
  letter-spacing: 2px;
`;

const StyledHeader = styled.header`
  padding: 2rem 0;
  background-color: #f8f9fa;
  border-bottom: 1px solid #e9ecef;
  width: 100%;
`;

const StyledSubtitle = styled.p`
  color: #6c757d;
  text-align: center;
  font-size: 1.1rem;
  margin-top: 0.5rem;
`;

const HeaderContainer = styled.div`
  max-width: 1200px;
  margin: 0 auto;
  padding: 0 1rem;
`;

function Header() {
  return (
    <div>
      <h1>Gestor de Tareas</h1>
      <p>Bienvenido a tu aplicación de tareas en React 2025</p>
    </div>
  );
}

export default Header;

export default Header;

Explicación:

  • Importamos styled desde @emotion/styled.
  • Definimos StyledTitle usando styled.h1 con estilos CSS dentro de template literals.
  • Los estilos están encapsulados y solo afectan a este componente.
  • Usamos StyledTitle como un componente React normal.

Traducción a TailwindCSS (para esta página):

3. Creación de Div Círculo con Texto Dentro Usando Styled

Creando un Círculo Estilizado

Creamos un componente que muestra un círculo con un número dentro para representar tareas completadas en TaskCounter. Modifica src/components/TaskCounter.jsx:

// src/components/TaskCounter.jsx
import { useState, useEffect } from 'react';
import styled from '@emotion/styled';

const Circle = styled.div`
  width: 60px;
  height: 60px;
  background-color: #3498db;
  border-radius: 50%;
  display: flex;
  justify-content: center;
  align-items: center;
  color: white;
  font-size: 1.5rem;
  font-weight: bold;
  margin: 0 auto 1rem;
`;

function TaskCounter() {
  const [completedTasks, setCompletedTasks] = useState(0);

  useEffect(() => {
    console.log('Componente montado');
    setTimeout(() => {
      setCompletedTasks(5);
    }, 1000);

    return () => {
      console.log('Componente desmontado');
      localStorage.setItem('completedTasks', completedTasks);
    };
  }, []);

  useEffect(() => {
    console.log('El contador de tareas ha cambiado:', completedTasks);
  }, [completedTasks]);

  const handleIncrement = () => {
    setCompletedTasks(completedTasks + 1);
  };

  return (
    <div>
      <h2>Tareas Completadas:</h2>
      <Circle>{completedTasks}</Circle>
      <button onClick={handleIncrement}>
        Marcar Tarea como Completada
      </button>
    </div>
  );
}

export default TaskCounter;

Explicación:

  • Definimos Circle con styled.div, usando CSS para crear un círculo (border-radius: 50%) y centrar el texto (display: flex).
  • El número de tareas completadas se muestra dentro del círculo.

Traducción a TailwindCSS:

    5. Creación Input con Styled

    Estilizando un Input

    Estilizamos el input en TaskForm para un diseño atractivo. Modifica src/components/TaskForm.jsx:

    // src/components/TaskForm.jsx
import styled from '@emotion/styled';
import { useState } from 'react';

const Form = styled.form`
  display: flex;
  flex-direction: column;
  gap: 1rem;
  max-width: 400px;
  margin: 0 auto;
`;

const StyledInput = styled.input`
  padding: 0.75rem;
  font-size: 1rem;
  border: 2px solid #3498db;
  border-radius: 5px;
  outline: none;
  transition: border-color 0.3s ease;

  &:focus {
    border-color: #2980b9;
  }

  &::placeholder {
    color: #95a5a6;
  }
`;

const Button = styled.button`
  padding: 0.75rem;
  font-size: 1rem;
  background-color: #2ecc71;
  color: white;
  border: none;
  border-radius: 5px;
  cursor: pointer;
  transition: background-color 0.3s ease;

  &:hover {
    background-color: #27ae60;
  }
`;

function TaskForm({ onAddTask }) {
  const [taskText, setTaskText] = useState('');

  const handleSubmit = (e) => {
    e.preventDefault();
    if (taskText.trim()) {
      onAddTask(taskText);
      setTaskText('');
    }
  };

  return (
    <div className="task-form-container">
      <Form onSubmit={handleSubmit}>
        <h2 className="text-xl font-bold mb-4">
          Agregar Nueva Tarea
        </h2>
        <StyledInput
          type="text"
          value={taskText}
          onChange={(e) => setTaskText(e.target.value)}
          placeholder="Escribe una tarea"
          className="task-input"
          required
        />
        <Button type="submit" className="submit-button">
          Agregar
        </Button>
      </Form>
    </div>
  );
}

export default TaskForm;

    6. Props en Styled

    Usando Props en Styled Components

    Puedes pasar props a un styled component para modificar estilos dinámicamente. Modificamos TaskItem para que el triángulo cambie de color según la prop completed.

    Modifica src/components/TaskList.jsx:

    // src/components/TaskList.jsx
import { useState } from 'react';
import TaskItem from './TaskItem';
import TaskForm from './TaskForm';
import Card from './Card';

function TaskList() {
  const [tasks, setTasks] = useState([
    { id: 1, text: 'Aprender React', completed: false },
    { id: 2, text: 'Hacer un proyecto', completed: true },
    { id: 3, text: 'Descansar', completed: false },
  ]);
  const [view, setView] = useState('list');

  const addTask = (taskText) => {
    const newTask = { id: tasks.length + 1, text: taskText, completed: false };
    setTasks([...tasks, newTask]);
    setView('list');
  };

  const EmptyMessage = () => (
    
    
      
    No hay tareas disponibles.
    
    
    
  );

  const TaskListView = () => (
    <div className="task-list">
      <h2 className="text-xl font-bold mb-4">Lista de Tareas</h2>
      <button
        className="bg-blue-500 text-white px-4 py-2 rounded mb-4"
        onClick={() => setView('form')}
      >
        Agregar Tarea
      </button>
      <ul className="space-y-2">
        {tasks.map((task) => (
          <TaskItem
            key={task.id}
            id={task.id}
            task={task.text}
            completed={task.completed}
          />
        ))}
      </ul>
    </div>
  );

  const renderView = () => {
    switch (view) {
      case 'list':
        return (
          <div className="container mx-auto px-4">
            <Card>
              {tasks.length === 0 ? <EmptyMessage /> : <TaskListView />}
            </Card>
          </div>
        );
      case 'form':
        return (
          <div className="container mx-auto px-4">
            <Card>
              <button
                className="bg-gray-500 text-white px-4 py-2 rounded mb-4"
                onClick={() => setView('list')}
              >
                Volver a la Lista
              </button>
              <TaskForm onAddTask={addTask} />
            </Card>
          </div>
        );
      default:
        return (
          <div className="text-center py-4">
            <p className="text-red-500">Vista no encontrada</p>
          </div>
        );
    }
  };

  return (
    <div className="task-list-container py-8">
      {renderView()}
    </div>
  );
}

export default TaskList;

    8. Temas en Styled

    Implementación de temas para cambiar entre modo claro y oscuro:

    // src/theme.js
export const lightTheme = {
  primary: '#3498db',
  background: '#ffffff',
  text: '#2c3e50',
};

export const darkTheme = {
  primary: '#2980b9',
  background: '#2c3e50',
  text: '#ecf0f1',
};

// App.jsx
import { ThemeProvider } from '@emotion/react';
import { useState } from 'react';
import { lightTheme, darkTheme } from './theme';

function App() {
  const [isDark, setIsDark] = useState(false);
  const theme = isDark ? darkTheme : lightTheme;

  return (
    <ThemeProvider theme={theme}>
      <div className="app-container min-h-screen">
        <header className="app-header py-4">
          <div className="container mx-auto px-4">
            <button onClick={() => setIsDark(!isDark)}>
              {isDark ? 'Cambiar a tema claro' : 'Cambiar a tema oscuro'}
            </button>
          </div>
        </header>
        <main className="app-main">
          {/* resto de la aplicación */}
        </main>
      </div>
    </ThemeProvider>
  );
}

    16. Styled Components con Referencias

    Uso de refs con styled components:

    // src/components/AutoFocusInput.jsx
import { useRef, useEffect } from 'react';
import styled from '@emotion/styled';

const FocusInput = styled.input`
  padding: 0.5rem;
  border: 2px solid #ddd;
  border-radius: 4px;
  width: 100%;
  max-width: 300px;

  &:focus {
    border-color: #3498db;
    outline: none;
  }

function AutoFocusInput() {

  useEffect(() => {
    inputRef.current?.focus();

    <div className="auto-focus-input-container text-center py-4">
      <label htmlFor="autoFocusInput" className="block text-sm font-medium mb-2">
        Campo con Auto-Focus
      </label>
      <FocusInput
        ref={inputRef}
        id="autoFocusInput"
        placeholder="Me enfoco automáticamente"
        className="auto-focus-input"
      />
    </div>
  );
}

export default AutoFocusInput;
