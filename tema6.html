Instalación de Material UI

¿Qué es? MUI es una librería de componentes React listos para producción, con sistema de temas, accesibilidad y diseño consistente.

¿Cómo funciona? Todos los componentes se importan desde @mui/material o paquetes específicos. Se envuelven en un ThemeProvider para controlar colores, tipografía y espaciado. 

# Crear proyecto
npx create-react-app mi-app --template cra-template-pwa
cd mi-app

# Instalar MUI núcleo + iconos
npm i @mui/material @emotion/react @emotion/styled @mui/icons-material

# Extras usados en este tutorial
npm i @mui/x-data-grid @mui/x-date-pickers dayjs

// src/main.jsx o src/index.jsx
import React from "react";
import ReactDOM from "react-dom/client";
import { ThemeProvider, createTheme, CssBaseline } from "@mui/material";

const theme = createTheme({
  palette: { mode: "light", primary: { main: "#1e40af" }, secondary: { main: "#0ea5e9" } },
  shape: { borderRadius: 12 },
  typography: { fontFamily: ["Inter","system-ui","-apple-system","Segoe UI","Roboto","Ubuntu","Cantarell"].join(",") }
});

function AppWrapper() {
  return (
    
      
      {/* Tu App */}
    
  );
}

ReactDOM.createRoot(document.getElementById("root")).render(<AppWrapper />);

Explicación: createTheme construye un tema con paleta y tipografía. CssBaseline normaliza estilos. Todo componente MUI descendiente leerá el tema vía React Context.

Button, Icons y Stack

Concepto: <Button/> ofrece variantes (contained, outlined, text); @mui/icons-material provee íconos SVG; <Stack/> organiza elementos en fila/columna con espaciado responsivo. 

import { Button, Stack } from "@mui/material";
import Save from "@mui/icons-material/Save";
import Delete from "@mui/icons-material/Delete";

export default function ButtonsDemo() {
  return (
    <Stack direction={{ xs: "column", sm: "row" }} spacing={2}>
      <Button variant="contained" startIcon={<Save />}>Guardar</Button>
      <Button variant="outlined" color="secondary" endIcon={<Delete />}>Eliminar</Button>
      <Button variant="text" href="https://mui.com" target="_blank">Docs</Button>
    </Stack>
  );
}

Cómo funciona: startIcon/endIcon inyectan SVG. Stack usa direction y spacing para layout responsive.

List, ListSubheader, ListItemButton, ListItemIcon

Listas navegables y con iconos; ideales para menús.

import { List, ListItemButton, ListItemIcon, ListItemText, ListSubheader } from "@mui/material";
import Inbox from "@mui/icons-material/Inbox";
import Send from "@mui/icons-material/Send";

export function MailList({ onSelect }) {
  return (
    <List
      subheader={<ListSubheader component="div">Mensajes</ListSubheader>}
      sx={{ width: 300, bgcolor: "background.paper", borderRadius: 2 }}
    >
      <ListItemButton onClick={() => onSelect("inbox")}>
        <ListItemIcon><Inbox /></ListItemIcon>
        <ListItemText primary="Bandeja de entrada" secondary="12 nuevos" />
      </ListItemButton>
      <ListItemButton onClick={() => onSelect("sent")}>
        <ListItemIcon><Send /></ListItemIcon>
        <ListItemText primary="Enviados" />
      </ListItemButton>
    </List>
  );
}

La prop sx aplica estilos con el sistema de temas.

Drawer (menú lateral)

El Drawer muestra navegación lateral temporal o permanente.

import { useState } from "react";
import { Drawer, IconButton, Box, Typography } from "@mui/material";
import MenuIcon from "@mui/icons-material/Menu";

export default function DrawerDemo() {
  const [open, setOpen] = useState(false);
  return (
    <Box>
      <IconButton onClick={() => setOpen(true)} aria-label="abrir menú">
        <MenuIcon />
      </IconButton>

      <Drawer anchor="left" open={open} onClose={() => setOpen(false)}>
        <Box sx={{ width: 280, p: 2 }} role="presentation">
          <Typography variant="h6" gutterBottom>Navegación</Typography>
          {/* Aquí puedes renderizar <MailList/> del ejemplo anterior */}
        </Box>
      </Drawer>
    </Box>
  );
}

Controlado por estado open; anchor define el lado.

Tablas (DataGrid)

Tabla de alto rendimiento con paginación, orden y selección.

import { DataGrid } from "@mui/x-data-grid";

const columns = [
  { field: "id", headerName: "ID", width: 90 },
  { field: "name", headerName: "Nombre", flex: 1 },
  { field: "age", headerName: "Edad", width: 120, type: "number" },
];

const rows = [
  { id: 1, name: "Ana", age: 23 },
  { id: 2, name: "Luis", age: 31 },
];

export default function GridDemo() {
  return (
    <div style={{ height: 320, width: "100%" }}>
      <DataGrid
        rows={rows}
        columns={columns}
        pageSizeOptions={[5, 10]}
        initialState={{ pagination: { paginationModel: { pageSize: 5 }}}}
      />
    </div>
  );
}

Define columns (metadatos) y rows (datos). flex permite columnas fluidas.

Stepper, Step, StepLabel & Typography

Muestra progreso por pasos, útil en procesos largos.

import { useState } from "react";
import { Stepper, Step, StepLabel, Button, Typography, Box } from "@mui/material";

const steps = ["Cuenta", "Perfil", "Confirmación"];

export default function StepperDemo() {
  const [active, setActive] = useState(0);
  const next = () => setActive((p) => Math.min(p + 1, steps.length));
  const back = () => setActive((p) => Math.max(p - 1, 0));

  return (
    <Box>
      <Stepper activeStep={active}>
        {steps.map((label) => (
          <Step key={label}><StepLabel>{label}</StepLabel></Step>
        ))}
      </Stepper>

      <Typography sx={{ mt: 2 }}>Paso: {steps[active] || "Finalizado"}</Typography>
      <Box sx={{ mt: 2, display: "flex", gap: 1 }}>
        <Button onClick={back} disabled={active===0}>Atrás</Button>
        <Button variant="contained" onClick={next}>{active===steps.length-1? "Finalizar":"Siguiente"}</Button>
      </Box>
    </Box>
  );
}

Tabs & Box

Pestañas para organizar contenido; Box es un contenedor versátil con prop sx.

import { useState } from "react";
import { Tabs, Tab, Box, Typography } from "@mui/material";

export default function TabsDemo() {
  const [tab, setTab] = useState(0);
  return (
    <Box>
      <Tabs value={tab} onChange={(_,v) => setTab(v)}>
        <Tab label="Perfil" />
        <Tab label="Seguridad" />
      </Tabs>>
      {tab===0 && <Typography sx={{ p:2 }}>Contenido de Perfil</Typography>}
      {tab===1 && <Typography sx={{ p:2 }}>Opciones de Seguridad</Typography>}
    </Box>
  );
}

Dialog, DialogActions, DialogContent, DialogTitle, Slide

Diálogos modales para confirmar acciones o mostrar formularios.

import { useState, forwardRef } from "react";
import { Dialog, DialogTitle, DialogContent, DialogContentText, DialogActions, Button, Slide } from "@mui/material";

const Transition = forwardRef(function Transition(props, ref) {
  return <Slide direction="up" ref={ref} {...props} />;
});

export default function DialogDemo() {
  const [open, setOpen] = useState(false);
  return (
    <>
      <Button variant="contained" onClick={() => setOpen(true)}>Eliminar</Button>
      <Dialog open={open} TransitionComponent={Transition} onClose={() => setOpen(false)} keepMounted>
        <DialogTitle>Confirmar</DialogTitle>
        <DialogContent>
          <DialogContentText>¿Seguro que deseas eliminar el registro?</DialogContentText>
        </DialogContent>
        <DialogActions>
          <Button onClick={() => setOpen(false)}>Cancelar</Button>
          <Button color="error" variant="contained">Eliminar</Button>
        </DialogActions>
      </Dialog>
    </>
  );
}

Autocomplete & TextField

Entrada de texto con sugerencias, validación y estados.

import { useState } from "react";
import { Autocomplete, TextField, Stack } from "@mui/material";

const opciones = ["React", "Vue", "Angular", "Svelte"];

export default function AutoForm() {
  const [valor, setValor] = useState(null);
  const [email, setEmail] = useState("");
  const error = email && !/.+@.+\..+/.test(email);

  return (
    <Stack spacing={2}>
      <Autocomplete
        value={valor}
        onChange={(_, v) => setValor(v)}
        options={opciones}
        renderInput={(params) => <TextField {...params} label="Framework" />}
      />

      <TextField
        label="Correo"
        value={email}
        onChange={(e) => setEmail(e.target.value)}
        error={error}
        helperText={error ? "Correo no válido" : " "}
/>
    </Stack>
  );
}

DesktopDatePicker, TimePicker, LocalizationProvider

Selectores de fecha/hora con localización. Usaremos dayjs.

import { useState } from "react";
import dayjs from "dayjs";
import "dayjs/locale/es";
import { LocalizationProvider } from "@mui/x-date-pickers/LocalizationProvider";
import { AdapterDayjs } from "@mui/x-date-pickers/AdapterDayjs";
import { DesktopDatePicker, TimePicker } from "@mui/x-date-pickers";
import { Stack, TextField } from "@mui/material";

export default function PickersDemo() {
  const [fecha, setFecha] = useState(dayjs());
  const [hora, setHora] = useState(dayjs());

  return (
    <LocalizationProvider dateAdapter={AdapterDayjs} adapterLocale="es">
      <Stack spacing={2}>
        <DesktopDatePicker
          label="Fecha"
          value={fecha}
          onChange={(v) => setFecha(v)}
          slotProps={{ textField: { fullWidth: true } }}
        />
        <TimePicker
          label="Hora"
          value={hora}
          onChange={(v) => setHora(v)}
          slotProps={{ textField: { fullWidth: true } }}
        />
      </Stack>
    </LocalizationProvider>
  );
}

El proveedor de localización inyecta formato y traducciones a los pickers.
